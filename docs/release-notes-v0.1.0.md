# Royale Launcher v0.1.0

Первый публичный релиз десктопного лаунчера Royale.

## Что входит в релиз

- `RoyaleLauncherSetup.exe` - установщик для Windows
- `RoyaleLauncherPortable.exe` - портативная Windows-версия
- `1.21.11.zip` - текущий пакет клиента

## Основные возможности

- интерфейс в стиле game launcher с одной главной кнопкой установки и запуска
- фиксированный каталог версий от владельца проекта
- автоматическое сохранение настроек
- проверка обновлений лаунчера через GitHub Releases
- обновление клиента поверх существующей папки без лишнего дублирования
